# SQL Table
```sql
CREATE TABLE inventory (
    id INT AUTO_INCREMENT PRIMARY KEY,
	profile_pic VARCHAR(255),
    user_info VARCHAR(255),
    gender VARCHAR(10),
    item_name VARCHAR(255),
    quantity INT,
    price INT
);
```
